# Django Banking Web Application
The Banking Web Application is a web-based system that allows users to manage their bank accounts online. The application provides features such as account creation, login authentication, balance checking, and transaction management. This project demonstrates backend development using Python and Django along with database integration.

# Features
User Registration and Login Authentication
Secure Account Management
Deposit and Withdraw Money
View Account Balance
Transaction History
Admin Panel for User Management
